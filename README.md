## How to Use

1. Download `dataNerdsDashboardForUnity.html` or visit the [GitHub Pages site](https://datanerdsdash.github.io/dataNerdsDashboardForUnity/)
2. Open in your browser
3. Upload your Unity earnings export file(s)
4. Optionally upload a naming file + withdrawal file for full features

That's it. One file, no install, no account, no internet required.
